Sobre o Projeto

O Clima Espacial Kids é um projeto educativo que apresenta o clima espacial para crianças de forma divertida e interativa.

O projeto foi desenvolvido pela Equipe Bellatrix durante o NASA Space Apps Challenge 2025.
